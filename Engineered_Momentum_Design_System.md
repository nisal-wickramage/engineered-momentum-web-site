# Engineered Momentum Design System

## Brand Principles

-   Engineering first, marketing second
-   Minimal over decorative
-   Structured over artistic
-   High signal, low noise
-   Timeless over trendy

## Color Palette

  Role            Hex
  --------------- -----------
  Primary         `#0B132B`
  Background      `#FFFFFF`
  Accent          `#10B981`
  Neutral Light   `#F3F4F6`
  Neutral Dark    `#4B5563`

**Accent usage:** links, buttons, highlights, diagrams only.

## Typography

-   Font: Inter (preferred)
-   Weights: Bold, Semibold, Regular
-   No decorative fonts; avoid italics except quotations.

## Spacing

Use an 8-point spacing scale: `8, 16, 24, 32, 48, 64`

## Shapes

-   Rounded corners: 6--8 px
-   Minimal shadows
-   No gradients or glassmorphism

## Icons

Use one library consistently (Lucide recommended).

## Visual Style

Prefer: - Architecture diagrams - Sequence diagrams - Flowcharts - Code
snippets - Terminal screenshots

Avoid stock photography unless necessary.

## Diagram Rules

-   Same font
-   Same line thickness
-   Same colors
-   Consistent spacing
-   Left-to-right flow where possible

## Photography

If used: - Monochrome - Architectural - Infrastructure - Abstract
engineering

## Voice

-   Clear
-   Practical
-   Evidence-based
-   Systems thinking

## Slides

-   One idea per slide
-   Large heading
-   Minimal text
-   Diagram-first

## Blog Hero

    ENGINEERED MOMENTUM

    <Title>

    One-line subtitle

## YouTube Thumbnails

-   White background
-   3--5 word title
-   One simple diagram or icon
-   Small logo
-   One accent color

## Consistency Checklist

-   One font family
-   One accent color
-   8-point spacing
-   Diagram-first
-   Generous whitespace
